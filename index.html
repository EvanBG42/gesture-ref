Gesture Reference Sheet

Universal Gestures

ASL Task / Subtask Gesture

ALEXA (name sign)

Get Alexa's attention.

Using the A-handshape, draw an X in front of you.

STOP

Stop current task

Push forward an open palm

HOME

Go to the home screen

Draw a house, starting from the top of the roof
